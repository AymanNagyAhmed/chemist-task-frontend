## Prerequisites
- Node.js (v22.11.0)
- jest

## Frontend Installation
### Local Development Setup
1. Clone the repository
2. copy .env.dev to .env
3. npm install
4. npm test
5. npm run start:dev
6. visit "localhost:3000"
